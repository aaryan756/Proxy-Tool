# Aaryan's Proxy Tool

This proxy tool allows the user to scrape the proxy type of their choice.

## Installation

Use the package manager [pip](https://pip.pypa.io/en/stable/) to install foobar.

```bash
pip install requests, colorama
```

## Usage

```python
python3 main
```
